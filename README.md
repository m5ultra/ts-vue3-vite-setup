# vue3项目模板

### 安装Eslint、 CommitLint、 Prettier

#### 安装 prettier
```
yarn add prettier -D

```

根目录增加配置文件 .prettierrc
```
{
 "trailingComma": "es5",
 "tabWidth": 2,
 "semi": false,
 "singleQuote": true
}

```

#### 安装 Eslint 

```
yarn add eslint -D

npx eslint --init //初始化Eslint配置文件

```
